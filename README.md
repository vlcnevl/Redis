# Redis <br/>
<b>Redis Nedir?</b> <br/>
Redis, açık kaynaklı bir in-memory veri yapısı deposudur. Verileri hafızada key-value (anahtar-değer) çiftleri olarak depolar ve yüksek performanslı bir veritabanı olarak kullanılır.
